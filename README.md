A modern, student-friendly budgeting and shared utilities web app for hostel life.

Managing expenses in hostel life is often chaotic — roommates struggle to split bills, track payments, and maintain a monthly budget. HLUM solves this problem with a sleek, Mint-inspired UI that makes budgeting, bill-splitting, and reminders simple, visual, and fun.

Features
	•	Dashboard
	•	Total budget, spent, and remaining amounts
	•	Circular progress bars and category breakdown charts
	•	Recent bills and transactions
	•	Expenses
	•	Add, view, and filter category-wise expenses
	•	Animated expense list for a real-time feel
	•	Shared Bills
	•	Auto-splitting of bills among roommates
	•	Pending/Paid status badges
	•	Interactive bill management
	•	Reminders
	•	Track pending dues
	•	Color-coded sliding cards for clarity
	•	Mark payments as done
	•	UI/UX
	•	Mint-inspired color scheme
	•	Glassy cards, soft shadows, and hover animations
	•	Responsive layout for desktop and mobile

Technologies Used
	•	HTML5, CSS3, JavaScript
	•	Chart.js for interactive charts
	•	LocalStorage for persistent data
